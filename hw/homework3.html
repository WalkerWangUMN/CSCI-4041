1. Suppose that the array A[0 : m−1] is used as a hash table. It has n keys, where n > 0. It resolves collisions by chaining. Simple uniform hashing is in effect.

2. (10 points.) The procedure HASH-SEARCH takes a hash table T[0 : m−1] and a key k as its parameters. Each element of T is either a key or NIL. The procedure searches for k in T by probing. If it finds k, then it returns the index j where k appears in T. If it cannot find k, then it returns −1. (A procedure very much like HASH-SEARCH is discussed on pages 269–274 of Cormen.)

HASH-SEARCH(T, k) 
    i = 0 
    repeat 
        j = h(k, i) 
        if T[j] == k 
            return j 
        else 
            i = i + 1 
    until T[j] == NIL or i == m 
    return −1

Suppose that the hash function h uses quadratic probing. It is defined like this:

h(k, i) = (h′(k) + i²) mod m

The function h′ takes a key k as its parameter, and returns a nonnegative integer that is somehow computed from k. (A more general version of h is discussed on page 272 of Cormen.)
